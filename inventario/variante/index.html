    Obtener una Variante

    GET https://contifico.com/sistema/api/v1/variante/<ID>/

    Devuelve una variante con el <ID> solicitado.

    Ejemplo:

        {
        "nombre": "COLOR",
        "id": "MZN5wbo2fDqaxEO0",
        "valores": [
            {
                "id": "wQRYjbqYi10bLNgm",
                "valor": "NEGRO"
            },
            {
                "id": "GOjZdy8mT7XbJ4mz",
                "valor": "BLANCO"
            }
        ]
    },
    Parámetro Tipo Longitud Descripción
    id varchar 16 Identificador de la variante/valores en el sistema
    nombre varchar - nombre de la variante
    valor varchar - Valor que se encuentra asociado a cada variante en el sistema
