    Variante

    Consulta

    Obtener listado de Variantes

    GET https://api.contifico.com/sistema/api/v1/variante/

    Devuelve un listado con todas las variante creadas en el sistema

    Ejemplo:

    [
    {
        "nombre": "COLOR",
        "id": "MZN5wbo2fDqaxEO0",
        "valores": [
            {
                "id": "wQRYjbqYi10bLNgm",
                "valor": "NEGRO"
            },
            {
                "id": "GOjZdy8mT7XbJ4mz",
                "valor": "BLANCO"
            }
        ]
    },
    {
        "nombre": "TAMAÑO",
        "id": "0RZxgepgh9Md1pVM",
        "valores": [
            { 
                "id": "wJJYjbqYi10bLNgm",
                "valor": "13 pulg."
            },
            {
                "id": "PxBleXJwu45brNOR",
                "valor": "15.6 pulg."
            },
            {
                "id": "qx4LmavQtENe3NrK",
                "valor": "17 pulg."
            }
        ]
    },
    ...   
]
    Parámetro Tipo Longitud Descripción
    nombre varchar 100 nombre de la variante
    id varchar 16 Identificador de la variante/valores en el sistema
    valores objeto - Objeto contenedor de los valores de la variante.
    valor varchar 100 Valor que se encuentra asociado a cada variante en el sistema

    Query Parameters

    Parámetro Tipo Longitud Descripción
    fecha_inicial date Filtro para consultar las categorias desde la fecha de ultima modificacion.
    fecha_final date Filtro para consultar las categorias hasta la fecha de ultima modificacion.

    Obtener una Variante

    GET https://api.contifico.com/sistema/api/v1/variante/<ID>/

    Devuelve una variante con el <ID> solicitado.

    Ejemplo:

        {
        "nombre": "COLOR",
        "id": "MZN5wbo2fDqaxEO0",
        "valores": [
            {
                "id": "wQRYjbqYi10bLNgm",
                "valor": "NEGRO"
            },
            {
                "id": "GOjZdy8mT7XbJ4mz",
                "valor": "BLANCO"
            }
        ]
    },
    Parámetro Tipo Longitud Descripción
    id varchar 16 Identificador de la variante/valores en el sistema.
    nombre varchar 100 nombre de la variante.
    valores objeto - Objeto contenedor de los valores de la variante.
    valor varchar 100 Valor que se encuentra asociado a cada variante en el sistema.
