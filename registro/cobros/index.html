    Cobros

    Consulta

    GET https://contifico.com/sistema/api/v1/documento/<ID>/cobro/

    Creación

    Una vez creado el documento si desea agregarle cobros puede hacer uso del servicio:

    POST https://contifico.com/sistema/api/v1/documento/<ID>/cobro/

    cambiando el parámetro por el id del documento (devuelto al momento de crear el documento).

    El formato del json que se envia al endpoint es el siguiente:

    {
    "forma_cobro": "TC",
    "monto": "100.00",
    "fecha": "08/11/2017",
    "numero_tarjeta": null,
    "numero_cheque": "",
    "tipo_ping": "D"
}

    donde se aceptan los siguientes valores:

    forma_cobro: Representa el la forma como se realiza el cobro sus valores pueden ser:

    Valor Tipo
    EF Efectivo
    CH Cheque
    TC Tarjeta de Crédito

    numero_tarjeta (campo opcional): Representa el número de tarjeta de crédito y se guarda únicamente si el campo forma_cobro es tarjeta de crédito (TC)

    numero_cheque: Representa el número de cheque y se guarda únicamente si el campo forma_cobro es cheque (CH)

    tipo_ping: Representa la red de cobro con la que se hace el débito en la tarjeta y se guarda únicamente si el campo forma_cobro es tarjeta de crédito (TC)

    "D" -> Datafast "M" -> Medianet
