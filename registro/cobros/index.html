    Cobros

    Consulta

    Podemos consultar los cobros que tiene un documento de cliente por medio de la url:

    GET https://api.contifico.com/sistema/api/v1/documento/<ID>/cobro/

    Usando el id devuelto en la creación de documento.

    Creación

    Se permite registrar cobros a un documento de dos formas:

    1. En el momento de su creación enviando dentro de la sección de cobros un arreglo con las distintas formas

    2. Si el documento fué creado previamente haciendo uso del servicio:

      POST https://api.contifico.com/sistema/api/v1/documento/<ID>/cobro/

      cambiando el parámetro por el id del documento (devuelto al momento de crear el documento).

    Los tipos de cobro soportados vía API son:

    Valor Tipo
    EF Efectivo
    CH Cheque
    TC Tarjeta de Crédito
    TRA Transferencia

    Registrar cobro tarjeta de crédito

    {
    "forma_cobro": "TC",
    "monto": "100.00",
    "fecha": "08/11/2017",
    "tipo_ping": "D"
}
    Campo Descripción
    forma_cobro "TC" para tarjeta de crédito
    monto valor cobrado
    fecha fecha del cobro
    tipo_ping Representa la red de cobro con la que se hace el débito. "D" -> Datafast "M" -> Medianet

    Registrar cobro efectivo

    {
    "forma_cobro": "EF",
    "monto": "100.00",
    "fecha": "08/11/2017"
}
    Campo Descripción
    forma_cobro "EF" para efectivo
    monto valor cobrado
    fecha fecha del cobro

    Registrar cobro cheque

    {
    "forma_cobro": "CH",
    "numero_cheque" : "12345",
    "monto": "100.00",
    "fecha": "08/11/2017"
}
    Campo Descripción
    forma_cobro "CH" para cheque
    numero_cheque número de cheque utilizado
    monto valor cobrado
    fecha fecha del cobro

    Registrar cobro transferencia

    {
    "forma_cobro": "TRA",
    "monto": "100.00",
    "cuenta_bancaria_id" : "qRYjbPkzC10eLNgn",
    "numero_comprobante" : "TRANSFERENCIA 123",
    "fecha": "08/11/2017",

}
    Campo Descripción
    forma_cobro "TRA" para transferencia
    monto valor cobrado
    fecha fecha del cobro
    cuenta_bancaria_id id de la cuenta bancaria
    numero_comprobante fecha del cobro
