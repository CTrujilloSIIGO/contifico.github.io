    Centros de Costo

    Consulta

    Obtener listado de Centros de Costo

    GET https://contifico.com/sistema/api/v1/contabilidad/centro-costo/?tipo=T

    Devuelve un listado con todos los centros de costo creadas en el sistema

    Ejemplo:

    [
    {
        "id": "xidkjeuekdldiej",
        "nombre": "centro costo 1",
        "codigo": "1.1",
        "tipo"  : "T"
    },
    {
        "id": "kdiekjdleke",
        "nombre": "centro costo 2",
        "codigo": "1.2",
        "tipo"  : "T"
    }
]
    Parámetro Tipo Longitud Descripción
    id varchar 16 Identificador del centro de costo en el sistema
    codigo varchar 50 Código del centro de costo
    nombre varchar 300 Nombre del centro de costo
    tipo varchar 2 Tipo de centro de costo T:Transaccional G:Grupo
