    Crear un asiento manual

    POST https://contifico.com/sistema/api/v1/contabilidad/asiento/

    Se debe enviar las siguiente estructura json en el cuerpo del requerimiento

    Ejemplo:

    {
    "fecha" : "16/05/2017",
    "glosa" : "Asiento via API",
    "gasto_no_deducible": 0,
    "prefijo": "ASI",
    "detalles" : [
        {
            "cuenta_id" : "MZN5wbojDFXaxEO0",
            "valor" : 10,
            "tipo": "D", 
            "centro_costo_id" : "EGOjZdy8yUNbJ4mz"
        },
        {
            "cuenta_id" : "x5onPeEZ2cWap1vL",
            "valor" : 150,
            "tipo": "D",
            "centro_costo_id" : "EGOjZdy8yUNbJ4mz"
        },
        {
            "cuenta_id" : "EGOjZdy9LcNdJ4mz",
            "valor" : 100,
            "tipo": "H",
            "centro_costo_id" : ""
        },
        {
            "cuenta_id" : "qlwKe5z03h0a31RZ",
            "valor" : 150,
            "tipo": "H", 
            "centro_costo_id" : ""
        }
    ]
}
    Parámetro Tipo Longitud Descripción
    fecha char 10 Fecha de la transacción formato "dd/mm/yyyy"
    glosa longtext Glosa del asiento
    gasto_no_deducible boolean 1 Identifica si es deducible o no el asiento
    prefijo char 3 prefijo del asiento (ver tabla de prefijos)
    detalles objeto - Cuentas involucradas
    Parámetro Tipo Longitud Descripción
    cuenta_id varchar 16 Identificador de la cuenta que se va a utilizar en el asiento
    valor decimal 10,2 Valor del detalle
    tipo char 1 Tipo de detalle D:Debe H:Haber
    centro_costo_id varchar 16 Identificador del centro de costo
